📖 配置说明

插件归属规则 用于将某些消息的 token 统计归入特定插件名下，便于分析各插件消耗。

每条规则包含一个插件名称和一组匹配条件，消息必须同时满足所有条件才会被计入该插件。

可选条件：

  • role - 消息角色：user、assistant、system、function
  • name_contains - 消息的 name 字段包含指定关键词
  • content_startswith - 消息内容以指定字符串开头
  • content_contains - 消息内容包含指定字符串

示例：

plugins:
  - name: "示例插件A"
    match:
      role: "user"
      content_contains: "以下是系统自动检索的背景上下文"

💡 规则按列表顺序依次匹配，一旦匹配成功就不再检查后面的规则。保存后需重启 AstrBot 生效。
